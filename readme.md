# 1.项目描述
基于Spring Boot/cloud分布式敏捷开发架构设计

# 2.项目组织结构
``` 
demo
├──demo-common          -- 公用组件
├──demo-config          -- 配置中心
├──demo-api             -- API接口总线系统（zuul）
├──demo-upms            -- 通用用户管理
├──demo-cms             -- 内容管理系统
├────demo-web-admin   -- 管理平台
├──demo-auth            -- 用户管理
├──demo-paly            -- 支付系统
├──demo-wechat          -- 微信
├──demo-mq              -- 消息系统
├──demo-shop            -- 电子商城
```
